**Running and Compiling Instructions**

Clone the repository from GitHub

Open Android Studio

Select File → Open and navigate to the project folder

Wait for Gradle to sync automatically

Connect an Android device or start an emulator (API level 21 or higher recommended)

Press the green Run button (or Shift+F10) to build and launch the app


No external libraries or special setup are required beyond a standard Android Studio installation.
